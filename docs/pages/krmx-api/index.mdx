# Krmx API
Krmx provides implementations its protocol for server and clients written in TypeScript. The server implementation can be used in a NodeJS backend and the clients can be used in NodeJS or in a React frontend.

If you're looking for the SDK reference, you can use the [TypeScript SDK reference](/sdk/index.html) generated from the Krmx source code.

If you decide to build your own Krmx implementation of a server or client in a different language or framework, then feel free to add that implementation by [opening a pull request](https://github.com/simonkarman/krmx).

If you need help setting up a project with Krmx, then please following the instructions at [Getting Started](/getting-started).

## Server Implementation
The Krmx server implementation (`@krmx/server`) is a NodeJS backend server and is written in TypeScript. You can use the [API documentation](/krmx-api/server), [TypeScript SDK reference](/sdk/modules/Krmx_Server.html), or [source code](https://github.com/simonkarman/krmx/blob/main/server/src/server.ts) to learn more about the Server API.

- [Creating a new Krmx Server](/krmx-api/server#creating-a-server)
- [Listening to events](/krmx-api/server#events)
- [More...](/krmx-api/server)

## Client Implementation
The Krmx client implementation (`@krmx/client`) is a NodeJS client and is written in TypeScript. You can use the [API documentation](/krmx-api/client), [TypeScript SDK reference](/sdk/modules/Krmx_Client.html), or [source code](https://github.com/simonkarman/krmx/blob/main/client/src/client.) to learn more about the Client API.

- [Creating a new Krmx Client](/krmx-api/client#creating-a-client)
- [Listening to events](/krmx-api/client#events)
- [More...](/krmx-api/client)

## React Client Implementation
The Krmx React client implementation (`@krmx/client-react`) provides an external store that wraps the Krmx Client, so that it can be easily used in React and is written in TypeScript. You can use the [API documentation](/krmx-api/client-react), [TypeScript SDK reference](/sdk/modules/Krmx_Client_for_React.html), or [source code](https://github.com/simonkarman/krmx/blob/main/client-react/src/client.tsx) to learn more about the React Client API.

- [Creating a new Krmx React Client](/krmx-api/client-react#creating-a-new-krmx-react-client)
- [Using the useClient hook](/krmx-api/client-react#using-useclient-hook)
- [Using the client](/krmx-api/client-react#using-the-client)
- [More...](/krmx-api/client-react)

## Base Library
The Krmx Base library (`@krmx/base`) is a shared library that contains the base classes and interfaces used by the Krmx Server and Clients. You can use the [TypeScript SDK reference](/sdk/modules/Krmx_Base.html), or [source code](https://github.com/simonkarman/krmx/blob/main/base/src/) to learn more about the Krmx Base API.
